## 🌟 Sobre o Projeto

Este é meu portfólio profissional como **Analista de Dados**, desenvolvido com tecnologias modernas para apresentar minhas habilidades, projetos e experiência na área de dados. Meu site conta com design responsivo, tema claro/escuro e formulário funcional de contato.

### 🎯 Objetivo

Demonstrar minha expertise em análise de dados através de uma interface moderna e interativa, facilitando o contato com potenciais empregadores e colaboradores.

## ✨ Funcionalidades

- **🎨 Design Responsivo**: Otimizado para desktop, tablet e mobile
- **🌙 Tema Claro/Escuro**: Alternância automática baseada na preferência do sistema ou manual
- **📱 Navegação Suave**: Menu fixo com scroll suave entre seções
- **📧 Formulário de Contato**: Integrado com Formspree para envio de mensagens
- **🔗 Links Diretos**: Acesso rápido aos meus projetos no Power BI
- **⚡ Performance**: Carregamento rápido e otimizado

## 🛠️ Tecnologias Utilizadas

### Frontend
- **HTML5**: Estrutura semântica
- **CSS3**: Estilização avançada com Tailwind CSS
- **JavaScript**: Interatividade e funcionalidades dinâmicas
- **Tailwind CSS**: Framework CSS utilitário
- **Font Awesome**: Ícones vetoriais

### Ferramentas
- **Formspree**: Serviço de formulários para contato
- **GitHub Pages**: Hospedagem gratuita
- **VS Code**: Ambiente de desenvolvimento
